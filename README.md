# 3-Tier TCP/RPC Calculator

Αυτό το αποθετήριο περιέχει την υλοποίηση ενός κατανεμημένου συστήματος τριών επιπέδων (3-tier architecture) σε γλώσσα C. Το σύστημα επιτρέπει σε πολλαπλούς πελάτες να εκτελούν μαθηματικές πράξεις σε διανύσματα, αξιοποιώντας συνδυαστικά υποδοχές δικτύου (TCP Sockets) και το πρωτόκολλο Απομακρυσμένης Κλήσης Διαδικασιών (ONC RPC).

## 📐 Αρχιτεκτονική Συστήματος

Το σύστημα αποτελείται από τρία διακριτά υποσυστήματα:

1. **TCP Client (Πελάτης):** Η διεπαφή του χρήστη. Επικοινωνεί μέσω πρωτοκόλλου TCP (IPv4 / AF_INET) με τον ενδιάμεσο διακομιστή. Διαβάζει διαδραστικά τα δεδομένα από τον χρήστη, τα αποστέλλει και αναμένει το τελικό αποτέλεσμα.
2. **Concurrent TCP Server / RPC Client (Ενδιάμεσος Κόμβος - Gateway):** Λειτουργεί ως γέφυρα. 
   - Ως **TCP Server**, δέχεται αιτήματα από πολλαπλούς TCP Clients ταυτόχρονα (concurrency), εξυπηρετώντας τον καθένα σε ξεχωριστό νήμα (thread). 
   - Ως **RPC Client**, παραλαμβάνει τα δεδομένα των πελατών και καλεί τις αντίστοιχες απομακρυσμένες διαδικασίες (RPC stubs) για να προωθήσει τους υπολογισμούς στον τελικό διακομιστή.
3. **RPC Server (Υπολογιστικός Κόμβος):** Ο τελικός διακομιστής που υλοποιεί την πραγματική επιχειρησιακή λογική (business logic). Εκτελεί τους μαθηματικούς υπολογισμούς και επιστρέφει τα αποτελέσματα στον ενδιάμεσο διακομιστή.

## 🧮 Υποστηριζόμενες Λειτουργίες

Ο χρήστης μπορεί να εισάγει έναν πραγματικό αριθμό $a$ και ένα διάνυσμα ακεραίων $Y = (y_1, y_2, ..., y_n)$ μεταβλητού μήκους $n$. Το σύστημα υποστηρίζει τους εξής υπολογισμούς:

1. **Μέση Τιμή:** Υπολογισμός και επιστροφή της μέσης τιμής των στοιχείων του διανύσματος $Y$ (Επιστρέφει: Πραγματικό αριθμό - `float`).
2. **Μέγιστο & Ελάχιστο:** Εύρεση του μέγιστου και του ελάχιστου στοιχείου του διανύσματος $Y$ (Επιστρέφει: Πίνακα 2 ακεραίων - `int[2]`).
3. **Βαθμωτός Πολλαπλασιασμός:** Πολλαπλασιασμός του πραγματικού αριθμού $a$ με κάθε στοιχείο του διανύσματος $Y$ (Επιστρέφει: Διάνυσμα πραγματικών αριθμών μήκους $n$).

## 🛠️ Τεχνολογίες & Εργαλεία
* **Γλώσσα Προγραμματισμού:** C
* **Δικτύωση:** BSD Sockets (TCP/IP, AF_INET)
* **Απομακρυσμένη Κλήση:** ONC RPC (`rpcgen` utility)
* **Ταυτοχρονισμός:** POSIX Threads (`pthread`)

## 🚀 Οδηγίες Μεταγλώττισης και Εκτέλεσης

### Προαπαιτούμενα
Για τη μεταγλώττιση του κώδικα σε περιβάλλον Linux, απαιτείται η βιβλιοθήκη **TI-RPC** και το εργαλείο `rpcgen`. 
*(Σε διανομές όπως το Ubuntu/Debian εγκαθίστανται με `sudo apt install libtirpc-dev rpcbind`, ενώ σε Arch Linux με `sudo pacman -S libtirpc rpcbind`).*
